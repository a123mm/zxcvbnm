部署恢复教程

1.下载 Ubuntu.tar.gz，将文件移动到目录
/storage/emulated/0/xinhao/data/

2.打开 ZeroTermux，按下音量上键，选择备份/恢复选项
3.选择恢复，选中 Ubuntu.tar.gz
4.输入新容器名称：Ubuntu，等待自动恢复完成

5.再次按下音量上键，进入容器切换，选中 Ubuntu
6.重启 ZeroTermux，在终端输入下面命令
curl ; bash -c "$(curl -L gitee.com/mo2/linux/raw/2/2)"
7.选择 proot容器 → 当前已安装容器列表 → 第一个容器，点击启动，即可进入Ubuntu系统
